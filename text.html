Let's Learn Basic Text Formatting

If you want the text to look like this

Bold Text: Use the <b> tag.

If you want the text to look like this

Italic Text: Use the <i> tag.

If you want the text to look like this

Underlined Text: Use the <u> tag.

If you want the text to look like this

Highlighted Text: Use the <mark> tag.

If you want the text to look like this

Smaller Text: Use the <small> tag.

If you want the text to look like this

Strikethrough Text: Use the <del> tag.
